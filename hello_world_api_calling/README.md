# Hello World API using NestJS (TypeScript)

## Project Overview
This project is a simple backend application built using NestJS with TypeScript.
It demonstrates how to create a basic REST API endpoint and run it locally.

## API Details

### Endpoint
GET /hello

### URL
http://localhost:3000/hello

### Response
Hello World from NestJS API

## PostgreSQL Integration

- PostgreSQL version: 16
- Database name: hello_world_db
- ORM: TypeORM
- Connection: localhost:5432
